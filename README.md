# ToF Module Ideas

> English is the primary text in this repository. A German clone is available in `README_DE.md`.

Public idea and candidate repository for larger ToF module concepts that may later become real services, containers, workers, or chain-level building blocks.

This repository does **not** publish the full internal V7 runtime, its active system truth, or sensitive implementation depth.  
It exists as a reduced idea and translation space for module-scale concepts before they become real runtime elements.

## What this repository is

This repository is a structured space for:

- capturing module ideas
- describing likely chain position
- making boundaries visible
- preserving openness before implementation
- preparing later translation into V7

## What this repository is not

This repository is not:

- runtime truth
- Werkbank salvage output
- silent release planning
- full implementation
- production service documentation

## Core idea

Some module ideas are large enough to become their own structural unit.

This repository exists so those ideas do not stay as loose chat fragments and do not get pushed directly into runtime without clean framing.

## Public boundary

This repository is intentionally limited.

It does **not** expose:

- internal runtime mechanics
- production IDs or secrets
- full internal service maps
- active operator logic
- sensitive implementation depth

It is a public-safe idea frame, not internal runtime disclosure.
