# Chain Positioning

> English is the primary text. The German clone is `03_chain_positioning_DE.md`.

Module ideas in this repository may be large enough to influence or occupy a structural place in the chain.

## Minimal positioning rule

A module idea should say where it most likely belongs, even if that position is still uncertain.

## Example chain positions

- ingress
- edge
- gate
- governance
- safety
- analysis
- worker
- state
- retention
- archive
- vector
- llm
- output

## Note

Positioning here is still a candidate statement, not runtime truth.
