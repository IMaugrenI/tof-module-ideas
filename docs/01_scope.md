# Scope

> English is the primary text. The German clone is `01_scope_DE.md`.

## Purpose

This repository holds module-scale ideas before they become runtime reality.

## Allowed here

- module candidates
- chain-position ideas
- domain grouping
- rough interfaces
- boundary statements
- translation notes toward V7

## Not allowed here as truth claims

- production truth
- hidden release decisions
- runtime declarations
- fake implementation status
- silent conversion of ideas into approved architecture

## Minimal rule

Capture clearly.
Separate idea from implementation.
Preserve openness until translation is real.
