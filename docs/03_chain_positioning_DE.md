# Kettenpositionierung

> Der englische Primärtext ist `03_chain_positioning.md`. Diese Datei ist der deutsche Klon.

Modulideen in diesem Repository können groß genug sein, um einen strukturellen Platz in der Kette zu beeinflussen oder selbst einzunehmen.

## Minimale Positionsregel

Eine Modulidee soll angeben, wo sie am wahrscheinlichsten hingehört, auch wenn diese Position noch unsicher ist.

## Beispielhafte Kettenpositionen

- ingress
- edge
- gate
- governance
- safety
- analysis
- worker
- state
- retention
- archive
- vector
- llm
- output

## Hinweis

Die Positionierung hier ist weiterhin nur eine Kandidatenaussage, keine Runtime-Wahrheit.
