# Geltungsraum

> Der englische Primärtext ist `01_scope.md`. Diese Datei ist der deutsche Klon.

## Zweck

Dieses Repository hält modulgroße Ideen fest, bevor sie zu Runtime-Realität werden.

## Hier erlaubt

- Modulkandidaten
- Kettenpositions-Ideen
- Domänengruppierungen
- grobe Schnittstellen
- Grenzbeschreibungen
- Übersetzungsnotizen in Richtung V7

## Hier nicht als Wahrheitsbehauptung erlaubt

- Produktivwahrheit
- versteckte Freigabeentscheidungen
- Runtime-Deklarationen
- gefälschter Umsetzungsstatus
- stille Umwandlung von Ideen in freigegebene Architektur

## Minimalregel

Klar erfassen.
Idee von Umsetzung trennen.
Offenheit bewahren, bis Übersetzung real ist.
