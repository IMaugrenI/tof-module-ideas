# Module Template

> English is the primary text. The German clone is `02_module_template_DE.md`.

Use this shape for each module candidate.

## Template

```md
# Module Name

## Status
idea / candidate / translated / parked / rejected

## Purpose
What the module is for.

## Problem
What gap or pressure it answers.

## Likely chain position
Where it probably belongs in the chain.

## Inputs
What comes in.

## Outputs
What comes out.

## Boundaries
What it explicitly is not.

## Runtime form later
service / worker / api / helper / other

## Dependencies
What it likely depends on.

## Risks
Hidden coupling, privacy, governance, safety, drift.

## Translation path
How it could later move toward V7.
```
