# Modul-Template

> Der englische Primärtext ist `02_module_template.md`. Diese Datei ist der deutsche Klon.

Nutze diese Form für jeden Modulkandidaten.

## Template

```md
# Modulname

## Status
idea / candidate / translated / parked / rejected

## Zweck
Wofür das Modul da ist.

## Problem
Welche Lücke oder welcher Druck dadurch beantwortet wird.

## Wahrscheinliche Kettenposition
Wo es wahrscheinlich in die Kette gehört.

## Inputs
Was hineinkommt.

## Outputs
Was herauskommt.

## Grenzen
Was es ausdrücklich nicht ist.

## Spätere Runtime-Form
service / worker / api / helper / other

## Abhängigkeiten
Wovon es wahrscheinlich abhängt.

## Risiken
Versteckte Kopplung, Privacy, Governance, Safety, Drift.

## Übersetzungspfad
Wie es später in Richtung V7 überführt werden könnte.
```
